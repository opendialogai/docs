# Launching Your Application

This section outlines how to launch your conversational application.

### OpenDialog's WebChat

#### Publish the Scenario

To launch your conversational application using the OpenDialog WebChat on your website or application you first need to activate the scenario you want to launch. If you have not done so already, you can do this by going to the 'Scenarios' page within the Designer and clicking on the slider which will turn green when 'active'.

![Active and draft scenarios](<.gitbook/assets/image (214).png>)

#### Embed code

You then need the embed script for the specific scenario you would like to add to your website or application.&#x20;

To get this, head to the 'Interface Settings' page and click on the 'Get embed code' button at the bottom of the right side bar

![](<.gitbook/assets/image (467).png>)

The Webchat embed code pop-up will then appear. You then need to copy the embed code by clicking on the 'Copy to Clipboard' button.

Once you have the embed code you then need to add it to the relevant pages on your website or application.

This embed code should be added at the bottom of the `body` section of the page(s) for which the Webchat should appear.&#x20;
