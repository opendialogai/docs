# Getting Started

To get started on your OpenDialog conversational application journey you need access to an OpenDialog instance. As OpenDialog is open-source software this means either [installing it on your own machine](installing-opendialog/opendialog-on-your-machine/), or setting up on a [virtual server](installing-opendialog/deployment/). 

Once OpenDialog is up and running and you can log in, you can start building your application! We provide some simple guides to illustrate what is possible as well as an example application.
