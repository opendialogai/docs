# Installing OpenDialog

{% hint style="warning" %}
Warning - you are browsing the documentation for an older version of OpenDialog. Please check out the latest version. 
{% endhint %}

OpenDialog is an open-source application, written in PHP \(using the [Laravel](https://laravel.com/) framework\) with a [Vue.js](https://vuejs.org/) frontend. 

To learn OpenDialog and experiment with conversational applications you can [install it on your own machine](opendialog-on-your-machine/). To serve an OpenDialog application, you need to [set it up on a server](deployment/).
