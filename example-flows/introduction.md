# Introduction

The easiest way to get familiar with OpenDialog is to start playing around with some example flows or _conversational patterns_. 

In OpenDialog a conversational pattern is a particular way to take advantage of the different ways conversations can be structured so as to build more robust and reusable conversations. They are reusable strategies that you can deploy in any type of conversation. 

By relying on patterns you can create sophisticated conversational applications faster. If you come across a pattern you think is useful to add to the list here [please let us know](https://opendialog.ai/contact-us/) - we are more than happy to credit its inventor!
