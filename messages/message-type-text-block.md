# Message type: Text block

A text message is a plain message with only text. You can add as many text blocks to your message as required. If you want to include a link, you can do this by pasting in the full URL, this will then be shown as a clickable link once displayed in OpenDialog's Webchat. &#x20;

![Message editor: Text messages](<../.gitbook/assets/image (308).png>)

There is no support at the moment for any other formatting such as hyperlinking text, bold, italics etc.

{% hint style="success" %}
**Saves Message: **Always remember to hit 'Save Message' before closing or navigating away from the edit screen.
{% endhint %}
