# Message type: Text block

A text message is a plain message with only text. You can add as many text blocks to your message as required. If you want to include a link, you can do this by pasting the full URL, this will then be shown as a clickable link once displayed in webchat.  

![Message editor: Text messages](../.gitbook/assets/image%20%28302%29.png)

There is no support at the moment for any other formatting such as hyperlinking text, bold, italics etc.
