# Message type: Custom Message

The custom message type allows you to add custom messages which are not currently available through the message editor UI. You can find a full list of supported messages not available via the UI through "Custom Message" and an explanation of the underlying XML structure [here](../developing-with-opendialog/message-markup.md).

To add a custom message; click on the 'Custom Message' block, and then click on the 'Message type' drop down to select the message type. This will then populate the XML snippet with a code snippet needed for that message type. You can repeat this to add multiple message types.
