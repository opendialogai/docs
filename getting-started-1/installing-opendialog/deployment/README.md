# OpenDialog Server Deployment

## System Requirements

### Browser Requirements

The OpenDialog web application should be functional in, all modern browsers that support CSS and JavaScript. We specifically test in the latest versions of:

* Chrome
* Safari
* Firefox

### Database Requirements&#x20;

#### SQL Database

OpenDialog is tested with _MySQL 5.7_ and _Postgress 12._&#x20;

#### Dgraph Database

OpenDialog is currently tested and works with _Dgraph v21.03.0_

### PHP Requirements

PHP 7.4

### Server Requirements

OpenDialog will work with any server that meets the PHP version requirements (including Nginx, Apache). We test using Nginx.
