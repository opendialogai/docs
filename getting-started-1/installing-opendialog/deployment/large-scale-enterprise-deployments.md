# Large-Scale Enterprise Deployments

For larger-scale deployments, where scalability, reliability and security are of critical importance and where the complete definition of infrastructure should be managed via code and is part of a continuous deployment process we make use of Terraform to create a dedicated Kubernetes cluster and deploy the OpenDialog application through Helm Charts.
